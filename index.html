What is a UI Kit?

A user interface (UI) kit is a collection of graphic files and resources that can help designers with the task of building UIs for their applications. UI kits typically include the user interface components that convey meaning and provide functionality to users – think buttons, widgets, checkboxes, progress bars, and navigation buttons.
